Документация маршрутов приложения
Основные маршруты

Публичные маршруты (доступны всем)
/ - Главная страница
/feedback - Страница отзывов
/about - Страница "О нас"
/training - Страница обучения
/login - Страница входа
/register - Страница регистрации
/reset-password - Страница сброса пароля
/course-info - Информация о курсе (модальное окно)
/listcourse - Список курсов
* - Страница 404 (не найдено)

Защищенные маршруты (требуют авторизации)
Все маршруты внутри /panel доступны только авторизованным пользователям.

/panel/course - Список курсов
/panel/course/:id - Конкретный курс
/panel/taskbook - Задачник
/panel/test - Тесты
/panel/roadmap - Навыки (дорожная карта)
/panel/cheatsheet - Шпаргалка
/panel/flashback - Флешбэк
/panel/mentorlist - Список менторов
/panel/mentorlist/:id - Профиль ментора
/panel/profile - Профиль пользователя
/panel/livecode - Интерактивная среда для написания кода
/panel/interview - Подготовка к техническим собеседованиям
/panel/projects - Раздел для демонстрации проектов 


Административные маршруты (требуют прав администратора)

/panel/admin-dashboard - Административная панель
/panel/admin/user/:id - Детали пользователя (для администратора)
